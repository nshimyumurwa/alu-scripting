# API Advanced

This project contains tasks for working with the Reddit API.

## Description

Learning to work with APIs, specifically:
- Querying the Reddit API
- Using pagination
- Parsing JSON results
- Making recursive API calls
- Sorting dictionaries by value

## Files

- `0-subs.py` - Returns the number of subscribers for a given subreddit
- `1-top_ten.py` - Prints the top 10 hot posts for a given subreddit
- `2-recurse.py` - Recursively returns all hot post titles for a subreddit
- `3-count.py` - Recursively counts and prints keyword occurrences

## Requirements

- Python 3.4.3+
- requests module
- PEP 8 style

## Usage
```bash
python3 0-subs.py programming
python3 1-top_ten.py programming
python3 2-recurse.py programming
python3 3-count.py programming 'python java javascript'
```
